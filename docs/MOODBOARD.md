## Design patterns that win in this industry

1. **Real product photography** — polished vessels, weld seams, in-shop scale shots. Never stock.
2. **Certifications above the fold** — ASME "U" / "R" stamp, ISO 9001, ISO 3834, AWS, PED. Critical trust signal.
3. **Capability-driven navigation** — organize by *what you can make* + materials/thickness ranges, not generic "Services".
4. **Spec tables & downloadable datasheets** — engineers want numbers fast.
5. **Persistent "Request a Quote" CTA** on every page.
6. **Case studies** structured problem → engineering → finished part.
7. **Numbers/stats band** — years, tonnage capacity, projects delivered, max vessel diameter.
