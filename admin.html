Publish your changes

Your edits are saved in this browser. To make them visible to everyone, download the two data files and commit them to the GitHub repository (they live in assets/data/).

How to publish (one time setup is easy):
1. Click the two download buttons above.
2. Replace assets/data/content.json and assets/data/news.json in your repo with the downloaded files.
3. Commit & push (or drag-drop them in GitHub's web UI). GitHub Pages updates automatically.
